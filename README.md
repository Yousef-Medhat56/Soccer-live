# Soccer Live

A website for football data: Getting matches on a specific date, Match Lineups, Match statistics, Match events, League standings and Teams.

## Technologies

1. NextJs
2. Tailwind css
3. Cheerio for web scraping

## Data source

The data is scraped from [btolat.com](https://www.btolat.com/matches).
